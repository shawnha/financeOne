Forecast 항목 등록 방법

1 내부계정을 "반복"으로 설정

먼저 해당 내부계정의 반복 속성을 켜야 합니다.

  1. 내부 계정 페이지 (localhost:3000/accounts) 접속
  2. 비용 트리에서 해당 계정 찾기 (예: "기타 구독", "AI SaaS")
  3. 계정 클릭 → 편집 → "반복" 체크 → 저장

반복으로 설정된 계정만 "전월 반복 가져오기"에 포함됩니다.

2 예상 탭에서 항목 추가

  1. 현금흐름 페이지 (localhost:3000/cashflow) 접속
  2. "예상" 탭 클릭
  3. 해당 월 선택
  4. 지출 섹션에서 "+ 항목 추가" 클릭
  5. 내부계정 선택 → 예상 금액 입력 → 결제방식 card 선택 → 저장

3 매월 초: "전월 반복 가져오기"

새로운 달이 시작되면:

  1. 현금흐름 → 예상 탭 → 해당 월 선택
  2. "전월 반복 가져오기" 버튼 클릭
  3. 반복 설정된 항목이 자동 복사됨
  4. 금액 변동이 있으면 수정

요약: 이것만 하면 됩니다

  1. 내부계정 11개를 "반복"으로 설정 (기타구독, AI SaaS, 출장, 광고비, 간식, 리테일식비, Admin구독, 주차/주유, api결제, 정수기, 가전렌탈)
  2. 4월 예상에 위 11개 항목 추가 + 금액 입력
  3. 스마트스토어(수입)도 반복 등록
  4. 비정기 대형 항목(통관, 퇴직금 등)은 확정 시 수동 등록

이렇게 하면 카드 커버리지 24% → 93%, 전체 forecast 정확도 대폭 개선.
