## 2026-06-07 — §13 결산 권위 cross-check (선수금·25300 해소)
결산 정본 = 각 법인 계정별원장(`~/Documents/HanahOneAll`, 시트명=계정코드). 읽기전용 분석, 커밋금지.

- **홀세일 26년 차트(48계정)**: 선수금=**25900**, 미지급금=**25300**, 미지급비용=26200(별개). 차량운반구20800·이자비용93100·단기차입금26000·장기차입금29300 보유.
- **홀세일 25300(미지급금) 원장 실내역**: 카드대금 클리어링(신한카드 식대/통행료/차량수리) + **현대캐피탈 ₩395,369/월 차량금융** 전부 25300 미지급금. → 현대캐피탈 차량할부=**미지급금**(이자/리스 분해 안 함).
