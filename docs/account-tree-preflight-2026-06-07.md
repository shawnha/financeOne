## ② gaap_mapping 역전 — 일원화 전 이관 필수 (review 적중)
생존(고사용) 코드가 gaap=0인데 폐기 레거시가 gaap=1을 쥔 경우 → 폐기하면 **US GAAP 연결에서 해당 계정 누락**.
- **차량유지비**: 생존 82200(95건) **gaap=0**, 레거시 51200·84500 gaap=1 → 이관.
- **통신비**: 50700(18·gaap1)·81400(4·**gaap0**)·82800(63·gaap1) — 3코드. canonical 자체를 결산권위로 정하고 gaap 이관.
